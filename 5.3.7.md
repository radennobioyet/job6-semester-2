# Pertanyaan

Di dalam method selection sort, terdapat baris program seperti di bawah ini:
```
int idxMin=i;
        for (int j=i+1; j<listMhs.length; j++){
            if (listMhs[j].ipk < listMhs[idxMin].ipk){
                idxMin=j;
            }
        }
```
Untuk apakah proses tersebut, jelaskan!

# Jawaban
Fungsi Proses
Proses tersebut digunakan untuk mencari indeks dengan nilai IPK terkecil (minimum) di dalam sisa array yang belum terurut.

Penjelasan Detail:
1. int idxMin = i;: Baris ini mengasumsikan bahwa elemen pada posisi awal perulangan (i) adalah nilai terkecil sementara.

2. Perulangan for (int j = i + 1; ...): Program akan menyisir elemen-elemen setelah i hingga akhir array untuk membandingkan nilainya.

3. Kondisi if (listMhs[j].ipk < listMhs[idxMin].ipk):

- Program membandingkan IPK pada indeks j dengan IPK pada indeks idxMin.

- Jika ditemukan IPK yang lebih kecil dari nilai minimum saat ini, maka...

4. idxMin = j;: Indeks minimum diperbarui ke posisi j.

Hasil Akhir: Setelah perulangan j selesai, variabel idxMin akan menyimpan lokasi (indeks) dari data dengan IPK paling rendah. Setelah proses ini selesai, biasanya kode akan dilanjutkan dengan menukarkan (swap) data pada listMhs[idxMin] dengan data pada listMhs[i].
